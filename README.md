# Sentiment Analysis and Topic Classification of Grab User Reviews Using IndoBERT and BERTopic

## Developer
Dinar Fadlilatunnisa 


## Overview

This project aims to analyze user reviews of the Grab application obtained from Google Play Store. The analysis consists of:

- **Sentiment Classification** (Negative, Neutral, Positive) using **IndoBERT**
- **Topic Modeling** using **BERTopic**
- **Topic Classification** using **IndoBERT** based on labels generated from BERTopic
- **Interactive Dashboard** developed with **Streamlit**

The project is intended to assist in understanding user opinions and identifying the main topics discussed in Grab application reviews, enabling service providers to evaluate and improve service quality.

---

## Features

- Data preprocessing for Indonesian text
- Sentiment classification using IndoBERT
- Topic extraction using BERTopic
- Topic classification using IndoBERT
- Model performance evaluation
- Interactive visualization with Streamlit
